如何学英语

发表于 2022-07-14

学习要有良好的反馈

如果一件事，要做很久很久,而且做这件事感觉不到任何变化，人会很迷茫很痛苦，人就很难把这件事坚持下去的；如果预期不久能学会，并且在学的过程中感觉这件事很有用、自己的进步很快，那人就会坚持去做这件事，生活也会变得有趣。

人是总是忘记，如果学习效率太低，那就会学一点忘一点，陷入苦海不能自拔。

如果人不能感觉到意义或乐趣，效率虽然高，但坚持不下来，最后总的结果也不会好。

学习一定指定好计划，不要随便拿着本书硬看，然后一日曝十日寒，不停地重复错误。

英语学习的方法

遗忘速度的理论

根据艾宾浩斯遗忘曲线及其他专家的总结（也不知道是谁，说的又对不对），遗忘的速度和一下要素有关：

  1. 记忆的材料是否重要、有意义、有趣；

  2. 分散的复习比集中复习更不容易忘记；

  3. “再认”的遗忘最少，“回忆”的遗忘最多；

  4. 对材料的熟悉程度，越熟悉的越不容易忘记；

根据以上要素，新背的单词应该重点复习。据称，分别间隔5分钟、30分钟、12小时、1天、2、4天、7天、15天这几个周期进行复习后，记忆就可以在人脑内长期保存。

背单词的方法

  1. 按照遗忘速度的理论和自身情况指定合适的学习计划。

  2. 前期对单词不熟悉时，放低标准，只需要能认出单词即可开始下一个单词，随着次数再提高对单词的熟悉程度。同理，前期也不该要求自己使用英文释义，使用中文释义即可。

  3. 据称，因为“前摄抑制”和“后摄抑制”的原因，最好把学习时间安排在早晨和晚上。

  4. 使用词根词缀记忆法、联想记忆法、典故记忆法、比较记忆法等方法。

目前的计划

考虑到以下原因：

  • 现在一天在公司的时间达到了11小时左右，产生了大量碎片时间，这些时间是很难使用词汇书学习的；

  • 手机app听单词的语音更加便捷，而且看风评似乎还不错；

目前打算使用手机app进行单词记忆，主要分晚上和上午2个时间进行学习，其他时候见缝插针。后续考虑购买专业书籍或上英文网站进行提升。

关于我给hexo更换主题失败这件事

发表于 2022-07-14

记录一下小白使用hexo犯的错误

我对hexo和github不是很熟,看了个教程就开始撸了.

网上搜到的像ctrl+f5和hexo clean ; hexo g ; hexo d;啥的都没起到作用.

后面才发现是brach一开始使用ph-pages,后面又改用master的原因.这样导致制造了2个分支,而一开始的ph-pages是默认分支,后面怎么改都访问的是一开始的页面,自然怎么改都没有作用了.

这种不了解node.js和git,直接照着教程做东西做东西一步小心陷坑里，浪费大量时间.还发现部署后,博客页面有时候没有变化,不知道是什么原因.
